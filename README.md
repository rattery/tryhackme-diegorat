## Organization

The repository is organized into numbered folders based on the order shown on the site (`1-example`, `2-example`, `3-example`, etc.).

Each folder contains:
- An overview file named after the folder in `.md` format
- The rooms from that section, numbered in the same order they appear on the site

I may organize it better in the future, but this structure works well for now.
